# TransitOps – Smart Transport Operations Platform

## Overview
TransitOps is a smart transport operations platform built using the MERN stack and TypeScript.

## Features
- Driver Management
- Vehicle Management
- Trip Management
- Maintenance Tracking
- Authentication
- Dashboard
- REST APIs

## Tech Stack
- React
- TypeScript
- Express.js
- Node.js
- MongoDB
- Vite

## Installation

```bash
npm install
npm run dev
```
